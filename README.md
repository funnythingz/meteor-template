Meteor template with TypeScript
=====

## Get Started

```shell
% npm install
```

### Directory

#### Dev

```shell
.ts ... src/**/*.ts
.scss ... src/sass/**/*.scss
```

> publish

```shell
.js ... public/app.js
.css ... public/css/styles.css
```

### Command

#### build

```shell
% grunt
```

#### watch

```shell
% grunt watch
```

#### Meteor

```shell
% cd public/
% meteor
```
